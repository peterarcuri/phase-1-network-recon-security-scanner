# Phase 1 — Network Recon Security Scanner

A Python-based network reconnaissance and security scanning toolkit focused on:

- TCP/IP networking fundamentals
- Port scanning
- Banner grabbing
- DNS reconnaissance
- TLS inspection
- Attack surface identification

---

# Objectives

This project is designed to strengthen practical DevSecOps and cybersecurity skills in:

- Networking
- Secure scripting
- Reconnaissance
- Security automation
- Service enumeration

---

# Features

## Current Features
- Multi-threaded TCP port scanner
- Banner grabbing
- Reverse DNS lookup
- CIDR subnet scanning
- JSON & CSV reporting

## Planned Features
- TLS certificate inspection
- UDP scanning
- Rate limiting
- Service fingerprinting
- Basic vulnerability detection

# Security Considerations

This tool is intended strictly for:

- educational purposes
- authorized environments
- defensive security testing

Unauthorized scanning of systems you do not own or have permission to assess is prohibited.
